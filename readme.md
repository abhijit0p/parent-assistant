# ⚡ Zero-Framework, High-Concurrency Multi-Tenant RAG Engine

A production-constrained, high-efficiency conversational AI assistant designed for educational institutions and sports academies. This system empowers parents to check active class attendance, view real-time outstanding dues, retrieve instant payment gateway links, and query institutional handbooks using an intelligent, local semantic retrieval layer.

Built entirely in native Node.js (ES Modules) without heavy framework abstractions (like LangChain or LlamaIndex) to maintain absolute algorithmic transparency over token payloads, multi-turn state mechanics, and tenant data isolation.

---

## 🏛️ Architecture & System Design

This project demonstrates a **Cloud-Local Hybrid Architecture** engineered to process natural language inputs, extract structured parameters, and isolate execution across independent data vectors.
